You may need to know this before you begin
Position an element by dragging
Adjust it by moving its control points
Delete it by using the delete key on your keyboard
'F' Key    brings an element to Front
'U' Key    increases(up) the opacity of an element
'L' Key    decreases(down) the opacity of an element
Color the element by using the panel in the top right corner. You may change the background too. The U and L key are very important for preparing a good creative work.
